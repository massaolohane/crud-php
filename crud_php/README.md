# Projeto CRUD em PHP e MySQL
Este projeto faz parte dos meus estudos em desenvolvimento web, com foco em backend e um pouco de integração frontend, e foi construído para aplicar na prática conteúdos como autenticação, sessões, segurança básica e operações CRUD.

## Funcionalidades

* Cadastro de usuários
* Login e logout
* Listagem de usuários
* Edição de usuários
* Exclusão de usuários
* Proteção de páginas com sessão
* Validações com JavaScript
* Dark Mode / Light Mode
* Interface responsiva

## Tecnologias Utilizadas

* PHP (procedural)
* MySQL
* MySQLi (prepared statements)

* HTML5
* CSS3
* JavaScript

### Ambiente de Desenvolvimento

* XAMPP
* Apache
* phpMyAdmin

### Controle de Versão

* Git
* GitHub

## Finalidade

Este projeto foi criado para fins de aprendizado, prática e construção de portfólio, com foco em desenvolvimento backend.

Sugestões e melhorias são bem-vindas!
